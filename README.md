🛠 Como Usar

 1. Clone este repositório:

        git clone https://github.com/mauricioarimoto/first-steps-dev.git

 2. Acesse a pasta desejada e execute os códigos com:

        gcc arquivo.c -o programa && ./programa  # Para C
